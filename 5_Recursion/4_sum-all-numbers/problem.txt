Sum of all numbers in array using Recursion


Write a function sum(n) that calculates the sum of all numbers in an array arr using recursion.
It sums from index 0 to n.


Concepts:
**********
1. Recursion: The function keeps summing the element at index n and calls itself with (n - 1)
2. Base Case: If n == 0, return the first element
3. Recursive Case: Return arr[n] + sum(n - 1)


Time & Space Complexity:
*************************
1. Time Complexity = O(n) - one recursive call per element
2. Space Complexity = O(n) - due to call stack


Examples:
**********
Input: [5, 2, 6, 1, 3]
Output: 17
// 5 + 2 + 6 + 1 + 3 = 17


Approach:
**********
1. If n == 0, return arr[0]
2. Otherwise, return arr[n] + sum(n - 1)
