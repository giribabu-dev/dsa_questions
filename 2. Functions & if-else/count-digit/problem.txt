Count the Number of Digits in an Integer

Problem Statement:
*******************
Write a function countDigits(n) that takes an integer n and returns how many digits it contains.


Requirements:
**************
1. Handles both positive and negative integers.
2. Return 1 if n is 0 (since 0 is a single digit number).


Examples:
**********
1. Input: 259 → Output: 3
2. Input: 1035 → Output: 4
3. Input: 0 → Output: 1


Approach:
**********
1. Handle Zero: If n==0, return 1 directly.
2. Covert to Positive: Use abs(n) to ignore sign.
3. Initialize a counter: Set count=0
4. Loop: While n>0
5. Divide n by 10 using integer division.
6. Increment count.
7. Return: The count after the loop finishes.


Dry Run:
********
n = 259
count = 0

259 > 0  => true
n = 25
count = 1

25 > 0  => true
n = 2
count = 2

2 > 0  => true
n = 0
count = 3

0 > 0  => false

Final: count = 3
